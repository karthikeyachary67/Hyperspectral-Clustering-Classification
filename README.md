# Hyperspectral Image Classification using Clustering

## Description
This project performs hyperspectral image classification using:
- K-Means
- Fuzzy C-Means
- Hierarchical Clustering

Dataset: Indian Pines

## Features
- No PCA used
- Preprocessing with normalization
- Hungarian algorithm for label matching

## Results
KMeans Accuracy: ~0.37  
FCM Accuracy: ~0.34  
Hierarchical Accuracy: ~0.37
